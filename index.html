🚀 欢迎使用 AI 教学设计生成器

本工具通过 AI 自动生成标准教学设计 HTML，支持全学科、各年级。

📋 使用前准备

你需要一个 AI API Key（调用 AI 的密钥）：

🔑 获取 API Key（推荐 DeepSeek）
1. 打开 platform.deepseek.com 注册登录
2. 左侧 → API Keys → 创建 API Key → 复制（以 sk- 开头）
3. 新用户免费送约 500 万 tokens，生成几百份教案都用不完
✅ 使用步骤：

① 双击同目录下的「启动生成器.bat」
自动启动本地服务器并打开此页面
② 填入 API Key → 选择模型 → 填写课题
③ 点击「生成教学设计」→ 下载/打印

💡 推荐使用 DeepSeek（免费额度，CORS 支持完善）
注册获取 Key：platform.deepseek.com

📝 AI 教学设计自助生成器

填写课题信息 → 一键生成标准教学设计 → 下载/打印

⚙️ API 设置 必填
▶
支持 OpenAI 兼容接口：DeepSeek / 通义千问 / 硅基流动 / Groq 等
新手推荐 DeepSeek （注册即送额度，无需绑定手机）
📋 教案信息
🧩 可选模块
模块 A （课程思政 / 德育渗透 / 核心素养）
模块 B （安全教育 / 实验规范 / 学科特色）

生成约需 30~60 秒 · 首次使用请先双击「启动生成器.bat」
