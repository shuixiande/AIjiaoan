## 重构为前后端分离架构

基于用户需求，将原本地单页 HTML 重构为 Express 前后端架构：

### 后端 (server.js)
- Express 服务器，端口 3000
- Cookie + localStorage 混合用户识别
- JSON 文件（data/users.json）存储用户使用记录
- 系统 API Key 存储在 config.json，不暴露给前端
- API 端点：
  - `GET /api/status` — 查询用户免费/付费次数
  - `POST /api/generate` — 使用系统 API 生成教案（扣减次数）
  - `POST /api/confirm-payment` — 用户确认付款（手动模式，+1 付费次数）
  - `POST /api/config` — 管理员配置系统 API
- 付费定价：￥2.99/次（config.json 可配置）

### 前端 (public/index.html)
- 参考原 index.html 的界面设计
- **双模式切换**：
  - 🏫 系统 API：首次免费 1 次，之后需付费
  - 🔧 自有 API Key：不限次数，浏览器直连 API
- **状态栏**：实时显示免费次数、付费次数
- **付费弹窗**：
  - 显示 qrcode.jpg 收款码
  - 展示价格 ￥2.99/次
  - 三步引导（扫码→支付→点击确认）

- **教案生成规则**：沿用 通用学科-教案生成规则.md 的 6 列表格规范
